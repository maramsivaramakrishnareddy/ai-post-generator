# AI Post Generator

This is a simple AI post generator project.

## Features
- Frontend: index.html
- Backend: FastAPI (main.py)
- Generates AI posts from a topic

## How to Run
1. Clone the repo
2. Navigate to backend folder and create virtual environment
3. Install dependencies: `pip install -r requirements.txt`
4. Add your API key in `.env` file
5. Run backend: `uvicorn main:app --reload --host 0.0.0.0 --port 8000`
6. Open `frontend/index.html` in browser
